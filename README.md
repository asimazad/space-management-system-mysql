# 🚀 Space Launch Management System

## 📌 Description
The Space Launch Management System is a SQL-based database project designed to manage and organize data related to space missions, rockets, launch sites, payloads, and space companies.

This system demonstrates the use of relational database design, primary and foreign key constraints, and structured queries to handle real-world scenarios in the space industry.

## 🎯 Objectives
- Store and manage space company data
- Track rockets and their payload capacities
- Manage launch sites and locations
- Record mission details and launch schedules
- Analyze mission and payload data using SQL queries

## ⚙️ Features
- Fully normalized relational database
- Multiple interconnected tables
- Data insertion and management
- Analytical queries (JOIN, GROUP BY, aggregation)
- Real-world inspired dataset

## 🛠️ Technologies Used
- SQL Server
- SQL (DDL, DML, Queries)

## 🚀 How to Use
1. Run `schema.sql` to create database and tables  
2. Run `data.sql` to insert sample data  
3. Run `queries.sql` to analyze and retrieve information  

## 📊 Example Queries
- Get all missions with company and rocket details  
- Count missions per rocket  
- Find latest missions  
- Analyze payload weights
